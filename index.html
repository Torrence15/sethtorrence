These bots helped me learn async and waiting for response before continuing. While writing in JavaScript I picked up on Node which introduced me to NPM. While researching NPM packages I discovered Electron a program that allows you to build standalone JavaScript programs that can display a canvas or an HTML website. This brings me to eventually switching to TypeScript for easier and more secure coding.
